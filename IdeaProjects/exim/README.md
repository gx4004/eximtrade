# Eximtrade — Website

**Live site:** https://eximtrade-deploy.vercel.app

---

## What is this?

This is the official website for **Eximtrade**, a Polish international trade & distribution company. It includes:

- Homepage with company overview
- About, Services, Industries, and Markets pages
- Contact page
- Legal pages (Privacy, Cookies, Terms)
- Available in **4 languages**: Polish, English, German, Russian

---

## How to open the site locally (no coding needed)

1. Download or clone this repository
2. Open the `src` folder
3. Double-click `index.html` — it opens in your browser

Or, to run it properly with a local server:

```
cd src
python3 -m http.server 8080
```

Then open **http://localhost:8080** in your browser.
